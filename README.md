하루동안 한 일을 기록하면 유용한 정보들을 시각화해서 보여주는 어플리케이션

## 개발 기간

2018년 4월~ 현재까지

## 기능

- 한 일, 시작시간, 종료시간, 카테고리, 활용시간 여부를 기록해서 등록한다.
- 하루 동안의 기록을 모아 유용한 정보를 시각화해서 보여준다.
- 일정 기간동안의 기록을 처리해서 유용한 정보로 시각화해준다.
